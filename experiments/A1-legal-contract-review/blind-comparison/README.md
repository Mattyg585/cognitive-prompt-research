# Blind Comparison — Contract Review Outputs

Three contract reviews of the same agreement (Common Paper CSA v2.1), each produced by a different AI prompt approach. The IDs are randomised.

## The Contract
A Cloud Service Agreement reviewed from the **customer/buyer** side. $150K/year SaaS platform, new vendor, mid-market deal. Data protection and IP ownership flagged as priorities.

## The Reviews
- `review-O08FZY.md`
- `review-8SNX69.md`
- `review-WLQZF0.md`

## What to evaluate
Read all three and rank them. Which one would you most want to receive if you were the lawyer or procurement lead walking into a negotiation? Specifically:

1. Which review gives you the best understanding of the contract's actual risks (not just a clause list)?
2. Which one best prepares you for the counterparty's likely positions?
3. Which one would you trust most to guide a real negotiation?
4. Are there findings in any review that the others missed entirely?
5. Which one reads most like a senior lawyer prepared it vs an automated checklist?

Rank them 1st, 2nd, 3rd with reasoning.
