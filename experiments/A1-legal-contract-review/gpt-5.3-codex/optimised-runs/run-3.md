## Proportional Clause Analysis

### Highest-materiality cluster (Data + IP)
- Sections 1.5, 1.6, 5.6, and 11.1 should be negotiated as one integrated package.
- Distinguish service operation rights from model-training rights.
- Prevent implicit expansion via undefined "related offerings" language.

### Security/governance cluster
- Make DPA execution contemporaneous with contract signing.
- Require baseline incident-notification timing and processor transparency.

### Commercial resilience cluster
- Add notice/cure discipline to suspension.
- Set explicit liability numbers and enhanced claim buckets.
- Tighten post-termination export/deletion mechanics.
