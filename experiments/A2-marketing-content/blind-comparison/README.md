# Blind Comparison — Marketing Blog Posts

Three blog posts announcing the same product launch (Relay — an async collaboration platform for distributed engineering teams). Each was produced by a different AI prompt approach. The IDs are randomised.

## The Brief
See `product-launch-brief.md` in this folder — a fictional product launch brief for Relay by Threadline. The task was to write a ~1200-1500 word launch blog post.

## The Blog Posts
- `blogpost-TD1GIJ.md`
- `blogpost-UPB876.md`
- `blogpost-UN2PS9.md`

## What to evaluate

Read all three as if you're the VP of Marketing at Threadline deciding which one to publish. Specifically:

1. Which blog post would you actually publish with minimal edits?
2. Which one has the strongest voice — sounds like a real company, not an AI?
3. Which one best uses the product details from the brief (beta stats, features, quotes) without reading like a spec sheet?
4. Which one would make a distributed engineering manager stop scrolling and actually read it?
5. Are there moments in any post that feel genuinely creative vs formulaic?

Rank them 1st, 2nd, 3rd with reasoning.

**Do NOT read KEY.md until after you've evaluated.**
