# Handoff Specification

## From: Angle Discovery → To: Channel Execution

### What crosses
- Structured Messaging Brief fields:
  - Audience & context
  - Core tension / job-to-be-done
  - Primary promise / transformation
  - Proof points
  - Voice/tone notes
  - CTA intent
  - Keywords (if provided)
  - Angle options (distinct framings)
- Any explicit constraints from the brief (brand, compliance, length)
- Selected channel(s)

### Format
- Labeled bullet list (no prose paragraphs)
- Compact, skimmable entries

### What gets dropped
- Exploratory reasoning, “why this is interesting” commentary
- Headline/hook formulas or SEO checklists
- Any ranking of angles

### Compression notes
- Keep each angle compact and factual
- Avoid long narrative explanations; preserve only the minimum needed for execution
