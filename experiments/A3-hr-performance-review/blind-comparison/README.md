# Blind Comparison: A3 — HR Performance Review

## Task

Write a manager performance review for Jordan Chen, Senior Software Engineer (IC4), based on the performance data in `test-material.md`.

## Evaluation Criteria

- **Depth**: Does the review go beyond surface-level observations? Does it identify patterns, tensions, and trajectory?
- **Specificity**: Are observations grounded in specific evidence from the performance data?
- **Completeness**: Does the review cover strengths, development areas, goal achievement, and forward-looking plans?
- **Audience Awareness**: Would this review be useful to Jordan, to calibration reviewers, and to HR? Does it serve the humans who will read it?
- **Bias and Fairness**: Does the review avoid problematic assumptions? Is it fair and evidence-based?
