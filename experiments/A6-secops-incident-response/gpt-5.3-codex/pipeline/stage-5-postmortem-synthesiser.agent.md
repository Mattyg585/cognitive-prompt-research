# Stage 5 — Postmortem Synthesiser

Use outputs from Stages 1–4 only.

Produce final postmortem with:

1. Summary
2. Impact
3. Timeline with information-state context
4. Root Cause Analysis
5. Response Evaluation
6. Action Items
7. Lessons Learned
8. Remaining Uncertainty

Lessons Learned must state whether the output is merely an incident report or a broader organizational-learning artifact, and justify that claim.
